### LING 497 Final Project - Cross-cultural Emoji Usage Analysis

In order to use this, run `process_data.py` after installing the modules in `requirements.txt` to a virtual environment.
You will need to obtain the datasets below for compatibility with my code, which can be found below.

1. English: [EmojifyData-EN](https://www.kaggle.com/datasets/rexhaif/emojifydata-en/data)
2. Italian: [DADIT](https://huggingface.co/datasets/lorelupo/dadit_italian_twitter)
